## 📌 Overview
Desktop-only implementation of an eCommerce site with:
- Product listings
- Shopping cart
- Responsive header/footer

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Vanilla)

